# 📈 Predictive Analytics Using Historical Per Capita Income Data

## Overview
This project uses Linear Regression to analyze historical China per capita income data and predict future income trends. It includes data preprocessing, exploratory data analysis (EDA), model evaluation, and forecasting.

## Objective
- Analyze historical per capita income data
- Build a Linear Regression model
- Evaluate model performance
- Forecast the next 5 years (2026–2030)

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Model Performance

| Metric | Value |
|--------|------:|
| R² Score | 0.9810 |
| MAE | 460.30 |
| RMSE | 603.70 |

## Visualizations
- Historical Trend
- Linear Regression Line
- Residual Plot
- 5-Year Forecast

## Conclusion
The Linear Regression model achieved an R² score of **0.9810**, indicating a strong fit to the historical data. The model was used to forecast China's per capita income for the next five years (2026–2030).
